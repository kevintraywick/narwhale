// Design Skills

20 commands.
Design fluency for AI.

Drop /polish, /audit, or any of the 20 skills directly into Cursor, Claude Code, Gemini CLI, Codex, Copilot, or Kiro. Each command encodes deep design expertise your AI would otherwise lack.

Diagnostic
Understand what's wrong before changing anything
2 commands
01 / AUDIT
🔬
/audit
Technical Quality Audit
Scores 5 dimensions with P0–P3 severity ratings and structured action plans. Catches accessibility failures, performance regressions, and responsive breakdowns before they ship.
02 / CRITIQUE
🎯
/critique
UX & Design Review
Scores against Nielsen's 10 heuristics, tests with persona archetypes, and assesses cognitive load. Surfaces hierarchy, clarity, and emotional resonance issues — not just visual ones.
Quality
Raise the baseline across every dimension
4 commands
03 / NORMALIZE
📐
/normalize
Align with Design System
Pulls rogue styles back into your design system's token and component vocabulary. Eliminates magic numbers, one-off colors, and inconsistent spacing.
04 / POLISH
✨
/polish
Final Pass Before Shipping
The last check — tightening spacing, sharpening type, smoothing transitions, and verifying visual rhythm. Run it last, right before you ship.
05 / OPTIMIZE
⚡
/optimize
Performance Improvements
Audits render cost, eliminates layout thrash, replaces heavy assets, and defers what doesn't need to block. Fast design is good design.
06 / HARDEN
🛡
/harden
Error Handling & Edge Cases
Handles empty states, loading states, long content, i18n text expansion, and error conditions. Makes your UI production-ready, not just demo-ready.
Intensity
Dial the boldness up or down
2 commands
07 / BOLDER
🔊
/bolder
Amplify Timid Designs
When a design is too safe, too gray, too hesitant — this makes it commit. Stronger hierarchy, more confident type, deliberate contrast. Pairs with /quieter.

08 / QUIETER
🔇
/quieter
Tone Down Bold Designs
When a design is shouting, this teaches it to whisper. Reduces visual noise, softens contrast, and creates breathing room for the content to speak.
pairs with /bolder
Adaptation
Reshape content and context
3 commands
09 / CLARIFY
💬
/clarify
Improve UX Copy
Rewrites labels, CTAs, error messages, and microcopy to be clearer, more actionable, and appropriately toned. Good copy is a design decision.
10 / DISTILL
💧
/distill
Strip to Essence
Removes everything that isn't essential. Formerly /simplify. If a UI feels cluttered, this finds the signal buried inside the noise.
11 / ADAPT
📱
/adapt
Different Devices & Contexts
Rethinks layouts and interactions for different viewport sizes, input modes, or usage contexts — not just scaling down, but genuinely reconsidering each breakpoint.
Enhancement
Add dimension, delight, and craft
7 commands
12 / TYPESET
Aa
/typeset
Fix Typography
Corrects font choices, size ramps, line-height, letter-spacing, and typographic hierarchy. Fixes the things that make text feel "off" without knowing exactly why.
13 / ARRANGE
⊞
/arrange
Fix Layout & Spacing
Adjusts grid, spacing rhythm, alignment, and visual grouping. When things feel slightly wrong but nothing specific stands out — this usually fixes it.
14 / ANIMATE
🎬
/animate
Add Purposeful Motion
Introduces transitions, micro-animations, and state changes that reinforce hierarchy and make interactions feel responsive — not decorative, purposeful.
15 / COLORIZE
🎨
/colorize
Introduce Strategic Color
Adds color with intention — semantic meaning, emotional tone, and brand alignment. Turns monochrome UIs into systems that communicate through hue.
16 / DELIGHT
🎉
/delight
Add Moments of Joy
Sprinkles personality — hover states, empty state illustrations, success animations, easter eggs. The things that make users smile and remember the product.
17 / ONBOARD
🚀
/onboard
Onboarding & Empty States
Designs first-run experiences and empty states that guide rather than confuse. Turns the blank-canvas problem into the product's first impression.
18 / OVERDRIVE beta
⚡
/overdrive
Technically Extraordinary Effects
Pushes the browser to its limits — WebGL, shader effects, procedural animation, physics-based motion. For when "impressive" is the actual requirement. Handle with care.
System
Set up and extract from your design system
2 commands
19 / TEACH
📚
/teach-impeccable
One-Time Project Setup
Gathers your project's design context — tokens, conventions, tone — and saves it to .impeccable.md. Run once; every subsequent command uses it to stay on-brand.
20 / EXTRACT
🧩
/extract
Create Design System Elements
Pulls repeated patterns out of one-off implementations and turns them into reusable components, tokens, or utilities. Turns good designs into scalable systems.
