// Workflow Commands

7 commands.
1 complete loop.

All commands use the /ce: prefix. Type them directly in your AI coding tool — or chain the whole pipeline automatically with /lfg.

01 / EXPLORE
🔭
/ce:ideate
Open-Ended Exploration
Generate options and angles before committing to a direction. For when you have a problem but no clear solution yet.
output → docs/brainstorms/
02 / CLARIFY
💬
/ce:brainstorm
Clarify Fuzzy Requirements
Turns vague feature ideas into scoped, unambiguous briefs through guided questioning. Nothing enters planning without a clear brief.
output → docs/brainstorms/
03 / PLAN
📐
/ce:plan
Structured Implementation Plan
Spawns three parallel research agents — codebase patterns, framework docs, industry standards — then synthesizes findings into a structured plan.
output → docs/plans/
agents → 3 parallel
04 / EXECUTE
⚡
/ce:work
Isolated Execution → PR
Sets up a git worktree, executes the plan step-by-step, runs tests and linting, then opens a pull request automatically. Step away while it works.
output → pull request
phases → isolation · exec · QA · PR
05 / REVIEW
🔍
/ce:review
14+ Specialized Review Agents
Deploys a fleet of reviewers simultaneously — security, performance, architecture, correctness, data integrity. Findings triaged as P1, P2, or P3.
agents → 14+ parallel
severity → P1 · P2 · P3
06 / COMPOUND
📈
/ce:compound
Teach the System What Worked
Six agents extract patterns from what just succeeded and update CLAUDE.md automatically — so every future task benefits from this one.
output → docs/solutions/ · CLAUDE.md
agents → 6 parallel
00 / FULL PIPELINE
🚀
/lfg
The Complete Loop, One Command
Describe an idea. Get a pull request. Chains all six stages automatically — ideate → brainstorm → plan → work → review → compound — spawning 50+ agents across every phase. You approve the plan, then step away.
chains → all 6 stages
agents → 50+ total
input → description only
